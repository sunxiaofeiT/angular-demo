## 代码运行方法
-  1、进入src目录（与docs同级的）。
-  2、使用npm install补上依赖包。也可以把之前备份的node_modules目录复制一份在项目根目录下。
-  3、使用json-server data.json启动后端服务。
-  4、使用npm start运行项目，打开浏览器访问http://localhost:4200/进入项目。
